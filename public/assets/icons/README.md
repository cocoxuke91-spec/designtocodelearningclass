# Icon assets

## Metrics

`metrics/` contains the four transparent glass 3D icons derived from the user-supplied four-icon reference sheet. They are used by the primary metric cards in the landing page.

## UI icons

The remaining interface icons are rendered as inline SVG paths in `app/LandingPage.tsx`. Their geometry follows the open-source Lucide icon style and is used under the Lucide ISC license.

Source: https://github.com/lucide-icons/lucide
